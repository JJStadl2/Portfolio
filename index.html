Intro

I am a former hospitality management specialist in the midst of a career change. In my past career I was exposed to various new applications that allowed me to do my job more effectively. Some of these applications worked well and others fell short. This got me interested in exploring ways to bridge the gap between developers and end users. Eventually this led to me being part of a team that tested new applications that were to be implemented company wide. In the process of testing, I was introduced to some basic SQL and HTML. It was very fascinating to me so, after 15 years in the industry I decided to change careers. I selected a degree in Computer Science because I feel there is no better means of improving the world around us than advancing technology. I also decided to pursue a concentration in Data Science because the amount of data available today is unprecedented and I feel the information that can be obtained from this data can be used to effectively implement emerging technologies.

Samples of My Work

Image Identifier

A machine learning application written in python using the Spyder IDE. It reads images from files and generates multiple models that allow the program to classify the image as belonging to one of several predefined classes. Creates graphics for selected features of images. Implements lasso regression, and random forest approaches.

Pandemic Tracker

A Python program written using Jupyter Notebook that simulates and tracks the spread of a virus in a sample population under various conditions. Test conditions include random contact between two members of the sample population, an immunity factor, the introduction of a vaccine, and variations of effectiveness of the vaccine. The program tracks the data and creates associated graphics.

War (The Card Game)

A program that simulates the card game War. Play occurs between one computer player and one human player. Functionality includes, shuffle, deal, draw, and quit. Consists of multiple classes and header files. Written in C++ using Visual Studio. Implements vectors, stacks, and pointers.

Schlep

This application is designed using JavaFX using the Netbeans IDE and is used to facilitate the delivery of large items to clients using a "sharing" model similar to Uber. The application follows MVC architecture, demonstrates how to connect to an external API, and stores client information in a secure database.
